# Visual Code 練習專案
****
這是一個 Visual Studio Code 的練習專案，包含 HTML、CSS 和 JavaScript 檔案。

## 專案結構

- `herofinishjsonphar.html` - 主要的 HTML 檔案
- `style.css` - 樣式表檔案
- `notjson.js` - JavaScript 檔案
- `桌面.code-workspace` - VS Code 工作區設定檔

## 如何使用

1. 用 VS Code 開啟工作區檔案 `桌面.code-workspace`
2. 在瀏覽器中開啟 `herofinishjsonphar.html` 來檢視結果
